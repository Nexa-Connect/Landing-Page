# Nexa Connect - Landing Page

Landing page da Nexa Connect, startup brasileira de Caruaru-PE especializada em desenvolvimento de software, inteligência artificial e automações.

## Tecnologias

- React 19 + Vite
- Ant Design
- CSS Modules
- Internacionalização (7 idiomas)
- Docker + Nginx

## Desenvolvimento

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build
```

## Docker

```bash
# Build e rodar
docker-compose up -d

# Ou direto com docker run
docker run -d -p 5173:5173 --restart always imagesnexaconnect/landing-page:latest
```
